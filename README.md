# SyntaxWear - Tênis e Sneakers Online

Este é um projeto de e-commerce para a venda de tênis e sneakers, desenvolvido com HTML e CSS.

## Descrição

O SyntaxWear é um site de e-commerce fictício com um design moderno e responsivo. A página principal apresenta um banner hero, categorias de produtos, uma grade de produtos em destaque e um rodapé com informações adicionais e links de navegação.

## Funcionalidades

- **Design Responsivo:** O layout se adapta a diferentes tamanhos de tela, proporcionando uma boa experiência de usuário em desktops, tablets e dispositivos móveis.
- **Navegação Intuitiva:** Menu de navegação com links para as principais categorias de produtos (Masculino, Feminino, Outlet), além de links de acesso rápido para "Nossas lojas" e "Sobre".
- **Seção de Categorias:** Apresenta as principais categorias de tênis (Casual, Esporte, Moderno, Futurista) de forma visualmente atraente.
- **Grade de Produtos:** Exibe os produtos em destaque em um layout de grade.
- **Rodapé Completo:** Inclui um formulário de inscrição para newsletter, links para redes sociais e um mapa do site com links de navegação adicionais.

## Tecnologias Utilizadas

- **HTML5:** Para a estruturação do conteúdo do site.
- **CSS3:** Para a estilização e o design do site, utilizando variáveis para um desenvolvimento mais modular e de fácil manutenção.

## Estrutura de Arquivos

```
/
|-- css/
|   |-- base.css
|   |-- reset.css
|   |-- variables.css
|   |-- components/
|       |-- footer.css
|       |-- header.css
|       |-- hero.css
|       |-- product-category.css
|       |-- product-grid.css
|-- images/
|   |-- banners/
|   |-- icons/
|   |-- logo/
|   |-- products/
|-- index.html
|-- README.md
```

- **css/:** Contém os arquivos de estilo do projeto.
  - **base.css:** Estilos base para o corpo da página e elementos comuns.
  - **reset.css:** Redefine os estilos padrão do navegador para garantir consistência.
  - **variables.css:** Define as variáveis de cores e fontes utilizadas no projeto.
  - **components/:** Contém os arquivos de estilo para os componentes específicos da página (cabeçalho, rodapé, etc.).
- **images/:** Contém todas as imagens utilizadas no site, organizadas em subpastas.
- **index.html:** O arquivo principal que estrutura a página inicial do site.
- **README.md:** Este arquivo, com a documentação do projeto.
